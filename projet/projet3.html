Nom : Outils & Scripts d’Automatisation

Description : Développement d’outils en Python, C++ et JavaScript permettant d’automatiser des tâches répétitives, optimiser des workflows, générer des assets, analyser des données ou accélérer la création de prototypes pour des projets plus complexes.

Exemples :

  • Script Python pour générer automatiquement des fichiers JSON
  • Outil C++ pour analyser des logs et détecter des erreurs
  • Mini‑programme JS pour créer des assets pixel‑art procéduraux

Objectif : Gagner du temps, améliorer la productivité et créer des outils réutilisables dans tes futurs projets.
