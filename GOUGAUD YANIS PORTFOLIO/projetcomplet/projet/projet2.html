> Projet Budget Familial

Application Flutter / Dart – Gestion locale du budget

Tableau de bord principal : aperçu des revenus, dépenses, solde et épargne.

Présentation du projet

Ce projet est une application mobile de gestion du budget familial, développée en Flutter (Dart). Elle permet de suivre toutes les dépenses et revenus du foyer, de manière simple, rapide et surtout totalement locale.

L’application est pensée pour un usage familial, mais avec un fonctionnement clair : seul l’administrateur du téléphone peut gérer le budget. Les autres membres peuvent être ajoutés pour attribuer les dépenses, mais ils n’ont pas accès à l’application directement.

J’ai également prévu une évolution future : permettre à chaque membre de la famille d’avoir son propre accès, tout en synchronisant les données entre les appareils. Pour l’instant, tout fonctionne en local, ce qui garantit une utilisation simple et sans connexion internet.

Fonctionnalités principales

  • Ajout de dépenses et revenus avec description, montant, catégorie et membre.
  • Dashboard complet : revenus, dépenses, solde, épargne.
  • Répartition par catégories (Logement, Divers, Enfants, Transport…).
  • Historique des dernières dépenses avec date et détails.
  • Interface simple et rapide pensée pour un usage quotidien.

Interface & expérience utilisateur

L’interface est construite autour d’un dashboard clair : chaque information importante est mise en avant avec un code couleur (vert pour les revenus, rose pour les dépenses, bleu pour le solde…).

L’ajout d’une dépense ou d’un revenu se fait via un bouton flottant qui ouvre une fenêtre dédiée : description, montant, catégorie, membre… Tout est pensé pour aller vite.
